# Linqs

**Linqs** — сокращатель ссылок с открытым исходным кодом.

## Стек технологий

- **Backend:** Go (pgx, golang-jwt, godotenv)
- **Frontend:** Vue 3 + TypeScript + Vite + Pinia + Vue Router + ECharts
- **База данных:** PostgreSQL + Redis
- **Инфраструктура:** Docker Compose (Nginx, multi-stage сборка)

## Функционал

### Пользователи и авторизация
- Регистрация по email или телефону
- Вход/выход с JWT-токенами
- Защита маршрутов

### Сокращение ссылок
- Создание коротких ссылок с авто-генерацией кода
- Поддержка кастомного короткого кода
- Редирект по короткой ссылке с трекингом
- Управление ссылками (список, удаление)

### Аналитика
- **Общая статистика:** всего кликов, уникальные клики, bounce rate, среднее время на сайте
- **Клики по дням:** график динамики переходов
- **География:** распределение по странам
- **Устройства:** десктоп vs мобайл, браузеры (Chrome, Firefox, Safari, Edge, Opera)
- **Bounce rate:** процент отказов по каждой ссылке
- Отслеживание сессий с хэшированием IP (SHA-256)

### Дополнительно
- Админ-панель пользователей
- Настройки профиля
- Cookie Consent Banner
- Адаптивный дизайн (Tailwind CSS)
- Полностью контейнеризированная сборка

## Развёртывание

### Важно

Nginx-конфигурация заточена под домен `linqs.ru`. При развёртывании на своём сервере **в первую очередь** замените `server_name` и связанные настройки в `url-shortener/frontend/nginx.conf` под ваш домен.

### Быстрый старт (локально)

```bash
chmod +x install.sh
./install.sh
```

Скрипт автоматически проверит зависимости, сгенерирует JWT-секрет, настроит окружение и запустит проект через Docker Compose.

> **Примечание:** работоспособность `install.sh` не проверена, рекомендуется ручная установка через `docker compose up -d`.

### Самостоятельный сервер (вручную)

```bash
cd url-shortener
docker-compose up -d
```

> **Примечание:** при самостоятельном развёртывании доступен **базовый функционал** сокращения ссылок и управления ими.

### Расширенный функционал

Полный набор функций, включая расширенную аналитику, приоритетную поддержку и дополнительные возможности, доступен исключительно по адресу [linqs.ru](https://linqs.ru).

## Назначение администратора

Первый администратор назначается вручную через PostgreSQL. Подключитесь к контейнеру БД и выполните:

```bash
# Войти в контейнер PostgreSQL
docker exec -it url-shortener-db psql -U appuser -d urlshortener

# Добавить колонку role, если её нет
ALTER TABLE users ADD COLUMN IF NOT EXISTS role VARCHAR(20) DEFAULT 'user';

# Назначить администратора (замените email на нужный)
UPDATE users SET role = 'admin' WHERE email = 'admin@example.com';
```
