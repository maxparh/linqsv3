Масштабируемое SaaS-приложение, построенное на современном стеке технологий. Платформа предоставляет мощные инструменты для управления проектами и детальной аналитики пользовательского поведения.
Технологический стек
Table
Компонент	Технология
Frontend	Vue.js 3 (Composition API, Vue Router, Pinia)
Backend	Go (Gin / Echo / Fiber)
База данных	PostgreSQL
Контейнеризация	Docker + Docker Compose
Авторизация	JWT-токены (доступ + обновление)
Основные возможности
🔐 Безопасность и авторизация

    Полноценная аутентификация на основе JWT-токенов (access + refresh)
    Защищённые маршруты API
    Автоматическое обновление сессии

📊 Продвинутая аналитика
Платформа собирает и визуализирует детальную статистику:
Table
Метрика	Описание
Клики	Отслеживание всех кликов пользователей по элементам интерфейса
Локация	Геолокация посетителей (страна, город, регион)
Устройства	Тип устройства, ОС, браузер, разрешение экрана
Интересы	Источник перехода (referrer) и целевые страницы выхода — полная воронка поведения пользователя
